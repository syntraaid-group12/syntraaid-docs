# SyntraAid Reporting and Impact Export Documentation

## Introduction

This document explains the reporting and export functionality within the SyntraAid platform.

The reporting system helps coordinators, administrators, and stakeholders track volunteer activities, attendance, events, and organizational impact through downloadable reports and dashboard analytics.

---

# Reporting Dashboard Overview

The reporting dashboard provides access to:

- Volunteer activity reports
- Attendance summaries
- Event participation reports
- KPI tracking metrics
- Notification statistics
- Exportable datasets
- Dashboard analytics

---

# Available Report Types

## Volunteer Activity Report

This report includes:

- Volunteer names
- Assigned tasks
- Task completion status
- Participation frequency
- Activity history

---

## Attendance Report

Attendance reports display:

- Event attendance records
- Check-in history
- Attendance percentages
- Volunteer participation trends

---

## Event Participation Report

This report tracks:

- Total event participants
- Volunteer engagement
- Event completion statistics
- Participation summaries

---

## KPI Dashboard Metrics

The KPI dashboard may include:

- Total volunteers onboarded
- Active volunteers
- Completed events
- Attendance performance
- Task completion rates
- Engagement statistics

---

# Export Functionality

Users can export reports in supported formats such as:

- PDF
- CSV

Export functionality allows stakeholders to:

- Share reports externally
- Archive records
- Analyze participation data
- Present impact summaries

---

# Export Workflow

## Exporting Reports

1. Open the Reporting Dashboard.
2. Select the preferred report type.
3. Apply filters if necessary.
4. Click the “Export” button.
5. Choose export format:
   - PDF
   - CSV
6. Download generated report.

---

# Troubleshooting Export Issues

## Common Issues

### Export Button Not Responding

Possible causes:

- Network issues
- Server processing delay
- Insufficient permissions

### Missing Report Data

Possible causes:

- Incorrect filters selected
- Incomplete attendance records
- Data synchronization delay
