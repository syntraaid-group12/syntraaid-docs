# SyntraAid Coordinator User Manual

## Introduction

Welcome to SyntraAid Coordinator Dashboard.

This manual explains how coordinators can manage volunteers, events, attendance, and communication within the platform.

---

# Coordinator Dashboard

The coordinator dashboard provides access to:

- Volunteer Management
- Event Coordination
- Attendance Monitoring
- Notifications
- Reports
- Profile Settings

---

# Managing Volunteers

## Viewing Volunteers

1. Open the “Volunteers” section.
2. View registered volunteers.
3. Click a volunteer profile to see details.

## Assigning Tasks

Coordinators can:

- Create tasks
- Assign volunteers
- Set deadlines
- Monitor progress

---

# Managing Events

## Creating Events

1. Open the “Events” section.
2. Click “Create Event”.
3. Enter event details.
4. Save changes.

## Updating Events

Coordinators can:

- Edit event information
- Change schedules
- Cancel events
- Notify volunteers

---

# Attendance Management

## Tracking Attendance

1. Open Attendance dashboard.
2. Select an event.
3. Review volunteer attendance records.

## Attendance Reports

The system allows coordinators to:

- View attendance summaries
- Monitor participation
- Export reports

---

# Notifications

Coordinators receive notifications about:

- Volunteer updates
- Event reminders
- Attendance submissions
- System alerts

---

# Profile Settings

Coordinators can update:

- Profile picture
- Contact details
- Password
- Notification preferences

---

# Troubleshooting

## Unable to Assign Tasks

- Check volunteer availability.
- Ensure event details are complete.

## Notification Issues

- Refresh the dashboard.
- Verify internet connection.

---

# Conclusion

The SyntraAid Coordinator Dashboard helps coordinators efficiently manage volunteers, attendance, and organizational activities.
